# CERN related information

## Web
CERN has many,many subdomains and domains. Some of these are :

### The .cern domain
https://home.cern Homepage of CERN \
https://lhc.cern Redirects to https://home.cern/science/accelerators/large-hadron-collider \
https://atlas.cern ATLAS functional homepage \
https://cms.cern CMS functional homepage \
https://alice.cern Alice functional homepage 

### The .cern.ch subdomain
https://info.cern.ch The first website on the world wide web \
https://cms.cern.ch CMS internal homepage; redirects to https://cms-info.web.cern.ch/ \
https://lhcb.cern.ch Redirects to https://lhcb.web.cern.ch 

### The .web.cern.ch subdomain
https://cms-info.web.cern.ch/ CMS internal site \
https://atlas.web.cern.ch Redirects to https://atlas.cern \
https://lhcb.web.cern.ch LHCb homepage
